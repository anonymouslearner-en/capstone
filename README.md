# Capstone Projekt

Dieses Projekt ist Teil des Capstone-Moduls im Fachbereich Elektrotechnik (F07) an der Technischen Hochschule Köln in Zusammenarbeit mit dem CAISA-Labor (F09).

**Projektthema:** Automatische Steuerung von Pflanzenanzuchtsystemen durch eine Platine

Dieses Repository ist eine Sammlung von Repos:

- **digital/** - Prototyp (Pio & Wokwi über VS Code)
- **hardware/** - Breadboard-Prototyp
- **main-codebase/** - Firmware-Code für die PCB

## Erste Schritte - Entwicklungsumgebung

### Voraussetzungen

Installieren Sie folgende VS Code-Erweiterungen:

- [PlatformIO IDE](https://marketplace.visualstudio.com/items?itemName=platformio.platformio-ide)
- [Wokwi Simulator](https://marketplace.visualstudio.com/items?itemName=wokwi.wokwi-vscode)

### Durchstarten

1. Öffnen Sie den Projektordner in VS Code
2. Erstellen und testen Sie über die PlatformIO-Symbolleiste oder Befehle
3. Verwenden Sie Wokwi zur Simulation mit dem digital/-Modul
